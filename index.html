Semanta Modeler

Get yourself a free database modeling tool!

Semanta Modeler is a free semantic modeling tool. It has proven it's value already for years, and is primarily aimed at database modelers. It gives you a professional semantic modeling tool, which generates scripts for multiple relational database types (e.g. Oracle, Microsoft SQL server, ...).

Semanta Modeler is provided, free of charge, by COAS Software Systems B.V.

You can download Semanta Modeler here. You can choose to download either the stable version, or a test (pre-release) version.
